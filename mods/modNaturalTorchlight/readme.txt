Natural Torchlight
4.31.1

Installation:
1) Copy this folder into your mods folder (NMM does this automatically);
2) Run Script Merger to identify and solve potential conflicts with other mods;
3) If installing the optional menu, copy the bin folder inside the mod into your game folder.

If you followed these instructions and still have problems, leave a comment on the mod page 
over at www.nexusmods.com where you likely downloaded this.
